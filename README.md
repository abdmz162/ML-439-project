# Setup and Installation

## 1. Clone the Repository

```bash
git clone git@github.com:abdmz162/ML-439-project.git
cd ML-439-project
```

---

## 2. Create Virtual Environment

### Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

### Windows

```powershell
python -m venv venv
venv\Scripts\activate
```

### macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3. Install Required Dependencies

```bash
pip install -r requirements.txt
```

---

# Running the Project

## Linux / macOS

```bash
python3 main.py
```

## Windows

```powershell
python main.py
```

---

# Deactivate Virtual Environment

```bash
deactivate
```
